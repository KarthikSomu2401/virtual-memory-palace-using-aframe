Welcome to Memory Palace with A-Frame tutorial


This project is an experimentation of learning a concept with the help of virtual reality environment. A-Frame is used to develop the environment with the help of some objects and images.

Please Read the Instructions before entering the VR Scene.


Enter Memory Palace Instructions

Basic Instructions

For computer screen, we can use WASD Controls

  • Use W to move forward in the scene
  • Use A to move left in the scene
  • Use S to move backward in the scene
  • Use D to move right in the scene
  • Click and drag to turn around the camera

For HMD, head movements tracks the movement in the scene


Inside the scene

  • Shop the ingredients in the supermarket
